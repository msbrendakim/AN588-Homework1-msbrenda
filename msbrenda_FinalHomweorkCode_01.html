First, create a vector of 1, word character strings comprising the first line of the novel Ulysses, by James Joyce: ‚ÄúStately, plump Buck Mulligan came from the stairhead, bearing a bowl of lather on which a mirror and a razor lay crossed.‚Äù Then, extract every third element of this vector.

library(stringr)

# Save the first line of the novel Ulysses to 'string'
string <- "Stately, plump Buck Mulligan came from the stairhead, bearing a bowl of lather on which a mirror and a razor lay crossed."

# Split the line into words
words <- str_split(string, " ")[[1]] 

# Extract every third element of this vector
every_third <- words[seq(3, length(words), by = 3)]
